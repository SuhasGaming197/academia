# 🌊 AQUADEMIA | Elite Academic Institution & 365-Day Biodome Waterpark Platform

> **Work Hard, Chill Hard**: An elite academic campus featuring a 365-day climate-controlled geodesic biodome waterpark, dry commuter canal water transit, and real-time Finite Element Analysis (FEA) physics engineering.

---

## 📌 Overview

**AQUADEMIA** addresses the ultimate student lifestyle question: *What if class finishes at 4:00 PM and you’re floating in a heated wave pool or sliding down a vertical drop by 4:15 PM?*

This repository contains the interactive site plan blueprint, 365-day climate control engine, commuter raft route calculator, and live physics simulations (FEA structural stress and Navier-Stokes fluid dynamics) designed to maximize student retention and well-being.

---

## 🚀 Key Features

### 🏛️ 1. Interactive Master Campus Map
- **Dual-Zone Layout**:
  - **Academic Zone**: Dormitories (D1-D3), Lecture Halls (L1-L6), Research Labs (R1-R4), Library, Student Center, Administration, Faculty Housing, Energy Center.
  - **Biodome & Waterpark Zone**: Retractable Glass Dome Enclosure, Wave Pool (3.5m deep), Surf Simulator, Vertical Drop Slide Tower (S1-S4), Resort Beach Area, Leisure Pools (1.5m deep), Dry Lockers.
- **Continuous Water Transit Loop**: Perimeter canal track with 5 active dock stations and animated floating commuter rafts.
- **Click-to-Inspect Nodes**: Click any building or pool to view capacity, ambient temperature, water depth, and travel stats.

### ☀️ 2. 365-Day Climate-Controlled Biodome Engine
- **Weather Scenario Selector**: Toggle between Sunny Summer (88°F), Autumn Downpour (52°F), Sub-Zero Winter Blizzard (14°F), and Extreme Heatwave (104°F).
- **Retractable Roof Control**: Interactive slider from 0% (Fully Sealed Geodesic Glass Dome) to 100% (Open Roof).
- **Subterranean Geothermal Water Pumps**: Holds wave pool water at a tropical 84°F year-round, regardless of outside freezing temperatures.

### ⚓ 3. "Lazy Commute" Transit Route Planner
- **Zero-Sweat Transit**: Compare **Dry Commuter Raft Float** vs. **Standard Walking**.
- **Onboard Deck Visualizer**: Raft features carpeted dry floors for backpacks, laptop desks, Wi-Fi, and coffee cup holders.

### 🔬 4. "Vibe Coding" FEA & Fluid Simulators
- **Geodesic Dome Structural FEA**: Calculates von Mises Stress ($\sigma_{\text{max}}$), Deflection (mm), and Safety Factor ($SF$) under wind shear (0-120 mph) and thermal gradients ($\Delta T$). Includes 1-click **FEniCS Python Code** export!
- **Navier-Stokes Hydrodynamics**: 2D velocity vector field grid simulating stream flow rate (m/s) and raft buoyancy.

### 📊 5. Student Life & Retention Analytics
- **KPI Metrics**: 99.4% Student Retention Rate, 99.8% Happiness Index, 3.86 Average GPA.
- **Timetable Walkthrough**: Interactive student daily schedule from 09:00 AM Quantum Mechanics to 04:15 PM Vertical Drop Slides!

## ⚡ Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/aquademia.git

# 2. Change directory
cd aquademia

# 3. Install dependencies
npm install

# 4. Start local development server
npm run dev
```

Open your browser to `http://localhost:3000`.
