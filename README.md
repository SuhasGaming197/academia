## 🚀 How to Download & Run the Website

Follow these 4 simple steps to run the application on your computer:

### 1️⃣ Download / Clone the Project
- **Option A (Download ZIP)**: Click the green **Code** button on GitHub and select **Download ZIP**. Unzip the folder onto your computer.
- **Option B (Clone via Git)**:
  ```bash
  git clone https://github.com/SuhasGaming197/academia.git
  cd academia
  ```

---

### 2️⃣ Open Terminal & Navigate to the Folder
Open **Terminal** (Mac) or **Command Prompt** (Windows) and enter the project folder:

```bash
cd path/to/academic-waterpark-campus
```

---

### 3️⃣ Install Dependencies
Run the following command to install the required packages:

```bash
npm install
```

---

### 4️⃣ Start the Website!
Run the development server command:

```bash
npm run dev
```

Once started, open your web browser (Chrome, Edge, or Safari) and go to:
👉 **[http://localhost:3000](http://localhost:3000)**

## 🏛️ Project Features & Platform Highlights

### 🗺️ 1. Interactive Master Campus Blueprint Map
- **Academic Zone**: Dormitories (D1-D3), Lecture Halls (L1-L6), Research Labs (R1-R4), Grand Library, Student Center, Administration, Energy Plant.
- **Biodome Waterpark Zone**: Retractable Glass Dome Enclosure, Wave Pool (3.5m deep), FlowRider Surf Simulator, Vertical Drop Slide Tower (S1-S4), Resort Beach, Leisure Pools, Dry Lockers.
- **Water Transit Loop**: Covered dry commuter canal with animated floating rafts moving between 5 active campus docks.

### ☀️ 2. 365-Day Climate-Controlled Biodome Controls
- Retractable roof simulator (0% Sealed to 100% Open).
- Weather engine scenarios: Sunny Summer (88°F), Rain (52°F), Sub-Zero Winter Blizzard (14°F), Heatwave (104°F).
- Geothermal heat pump telemetry keeping water at **84°F year-round**.

### ⚓ 3. "Lazy Commute" Transit Route Planner
- Compare **Dry Commuter Raft Float** (0 Cal effort, laptop desk, Wi-Fi, coffee) vs. **Standard Walking Path**.

### 🔬 4. "Vibe-Coding" FEA & Fluid Dynamics Simulators
- **Geodesic Dome Structural FEA**: von Mises Stress heatmaps, wind shear (0-120 mph), thermal expansion ($\Delta T$), and 1-click **FEniCS Python Code** export.
- **Hydrodynamics**: 2D Navier-Stokes velocity vector grid and raft buoyancy physics.
